# Finding Template

Standard format for vulnerability findings.

## JSON Schema (findings.json)

```json
{
  "metadata": {
    "skill": "skill-name",
    "agent": "agent-name",
    "target": "target",

  },
  "summary": {
    "total_findings": 3,
    "by_severity": {
      "critical": 1,
      "high": 1,
      "medium": 1,
      "low": 0
    }
  },
  "findings": [
    {
      "id": "finding-001",
      "title": "Reflected XSS in Search Parameter",
      "vulnerability_type": "Cross-Site Scripting (XSS)",
      "variant": "Reflected",

      "cvss": {
        "score": 7.1,
        "vector": "CVSS:3.1/AV:N/AC:L/PR:N/UI:R/S:C/C:L/I:L/A:N"
```

## Markdown Template (finding-NNN.md)

```markdown
# [Severity] Title

**Finding ID**: finding-NNN
**Discovered**: YYYY-MM-DD HH:MM:SS UTC
**Validated**: YYYY-MM-DD HH:MM:SS UTC

---

## Summary

[2-3 sentence vulnerability description]

---

## Severity Assessment

**CVSS Score**: X.X (Severity)
**CVSS Vector**: CVSS:3.1/AV:N/AC:L/PR:N/UI:R/S:C/C:L/I:L/A:N

**Mappings**:
- **CWE**: CWE-XX (Description)
- **OWASP**: AXX:2021 - Category

---

## Vulnerability Details

**Type**: Vulnerability Type
**Location**: https://example.com/path
**Parameter**: param_name

### Root Cause

[Technical explanation of why vulnerability exists]

---

## Proof of Concept

### Steps to Reproduce

1. Navigate to: URL
2. Enter payload: [payload]
3. Observe: [result]

### Visual Evidence

![Screenshot](evidence/screenshots/finding-NNN.png)

**Video**: [Demo](evidence/videos/finding-NNN.mp4)

---

## Impact Analysis

### Business Impact: Severity

- Session hijacking
- Account takeover
- Data theft

### Attack Scenario

1. Attacker action
2. Victim interaction
3. Payload execution
4. Impact achieved

---

## Remediation

### Immediate (Deploy within 24h)

[Quick fix with code example]

### Short-Term (1 week)

[Mitigation steps]

### Long-Term (Next sprint)

[Comprehensive solution with code examples]

---

## References

- [OWASP Guide](https://owasp.org/...)
- [CWE Entry](https://cwe.mitre.org/...)

---

*Finding generated by [skill-name] using [agent-name]*
```
